# Portfol.io

Конструктор художественного портфолио. Художник загружает фото работ, данные для маркировки (tombstone label по международному стандарту), биографию и CV — на выходе получает готовый PDF-портфолио для подачи на open call, кураторские гранты и конкурсы.

Идея из интервью с Ольгой Клип: маркетплейсы дают поля под международные стандарты, но ни один не позволяет выгрузить PDF, который нужен для заявок.

## Возможности
- Редактор слева, живой A4-превью справа
- Загрузка фото для каждой работы
- Музейная этикетка: название, год, материал/техника, размеры, серия, цена/статус
- Аннотация к каждой работе
- Artist statement + структурный CV (образование, выставки, награды)
- Экспорт в PDF (печать A4: обложка + statement/CV + страница на работу)
- Автосохранение в localStorage
- Онлайн-версия: один HTML-файл, заливается на любой хостинг

## Запуск
Открыть `index.html` в браузере. Зависимостей нет.

## Стек
Single-file HTML/CSS/JS. Без сборки и бэкенда (прототип).
